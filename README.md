# Kompot Bug Reports

Report bugs and issues for [Kompot](https://kompot.ai).

## How to Report

1. Click **[New Issue](../../issues/new/choose)**
2. Fill in the template
3. Submit

## What to Include

- **What happened** — describe the problem
- **Steps to reproduce** — how to trigger the bug
- **Expected behavior** — what should happen instead
- **Screenshots** — if applicable

## Status

We review issues regularly and will respond with:
- Questions if we need more info
- Confirmation when the bug is fixed
- Explanation if it's not a bug
