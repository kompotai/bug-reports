# Kompot Feedback

Report bugs, suggest improvements, and request features for [Kompot](https://kompot.ai).

## How to Submit

1. Click **[New Issue](../../issues/new/choose)**
2. Choose type: Bug Report, Improvement, or Feature Request
3. Fill in the template
4. Submit

## Issue Types

| Type | When to use |
|------|-------------|
| **Bug Report** | Something is broken or not working as expected |
| **Improvement** | Existing feature could work better |
| **Feature Request** | You need something that doesn't exist yet |

## What Happens Next

We review issues regularly and will respond with:
- Questions if we need more details
- Status update when work begins
- Notification when resolved
